# 愤怒的小鸟游戏开发计划

## 项目信息

**GitHub 仓库**: https://github.com/abcdefg-maker/PhaserGameDemo-AngryBird-DevelopingWithClaudeCode

**Git 远程地址**: git@github.com:abcdefg-maker/PhaserGameDemo-AngryBird-DevelopingWithClaudeCode.git

**当前版本**: v0.2.0 - 程序化图形生成系统

## 开发顺序

### 第一阶段：基础对象系统
1. ✅ **程序化图形生成系统** - 所有游戏对象的图形基础
   - 小鸟纹理（红鸟、黄鸟、炸弹鸟）
   - 方块纹理（木头、石头、玻璃，横竖方向）
   - 猪纹理（小、中、大）
   - 弹弓纹理
   - 地面纹理

2. **建筑方块系统**（木头、石头、玻璃）
   - 创建 Block 基类
   - 不同材质的物理属性（密度、摩擦力）
   - 耐久度系统

3. **猪系统**
   - 创建 Pig 类
   - 生命值系统
   - 不同大小的猪

### 第二阶段：核心玩法
4. **小鸟基类和红鸟**
   - 创建 Bird 基类
   - 红鸟实现（最基础）
   - 飞行物理

5. **弹弓系统**
   - 拖拽交互
   - 力度计算
   - 轨迹预测
   - 发射小鸟

6. **碰撞伤害计算**
   - 碰撞检测
   - 伤害计算公式
   - 对象销毁逻辑

### 第三阶段：扩展功能
7. **黄鸟和炸弹鸟**
   - 黄鸟加速能力
   - 炸弹鸟爆炸能力

8. **关卡数据和管理器**
   - 关卡数据结构
   - LevelManager
   - 关卡加载

9. **相机跟随系统**
   - 跟随小鸟飞行
   - 平滑过渡

10. **分数和星级系统**
    - 分数计算
    - 三星评级
    - UI显示

## 技术要点

### 物理引擎配置
- 重力：{ x: 0, y: 1.5 }
- 启用休眠优化
- Matter.js 物理引擎

### 材质属性参考
- **木头**：密度低，容易破坏
- **石头**：密度高，难以破坏
- **玻璃**：密度中，易碎

## 开发原则
- 每个功能独立开发和测试
- 循序渐进，避免错误积累
- 及时提交 Git 版本
